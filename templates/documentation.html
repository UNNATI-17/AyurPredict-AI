System Overview

AyurPredict is an AI-driven platform that predicts interactions between Ayurvedic herbs and biological targets, helping researchers and users discover potential therapeutic applications.

The system combines traditional Ayurvedic knowledge with modern machine learning techniques to provide evidence-based predictions for herb-target interactions, bioactivity strength, and therapeutic potential.

Key Innovation

AyurPredict bridges the gap between traditional medicine and computational biology by applying trustworthy machine learning models to predict herb-target interactions with high precision.

Model Selection & Optimization Strategy

Why Random Forest Was Chosen as the Best Model
Key Advantages
  • Highest Precision (0.5699): Most critical for trustworthy medical predictions
  • Best Prediction Stability (0.3075): Consistent and reliable outputs
  • Lowest Overfitting Gap (0.1229): Better generalization to new data
  • Robust Feature Importance: Clear interpretability of biological features
  • Handles Non-linear Relationships: Perfect for complex herb-target interactions
Trade-off Analysis
Accuracy vs. Trustworthiness: While Gradient Boost had slightly better R² (0.4681 vs 0.4487), Random Forest excelled in precision and stability - more crucial for healthcare applications.

Decision Rationale: In medical AI, a conservative and stable model that makes fewer but more reliable predictions is preferred over one with slightly better overall accuracy but higher variance.

Hyperparameter Optimization Process
Optimization Strategy
  • GridSearchCV & RandomizedSearchCV: Comprehensive parameter space exploration
  • Focus on Precision: Optimization targeted precision over raw accuracy
  • Biological Constraints: Parameters tuned to respect biological plausibility
  • Cross-Validation: 5-fold CV to ensure robustness
Key Optimized Parameters
  • n_estimators: Increased to 200 for better stability
  • max_depth: Limited to 15 to prevent overfitting
  • min_samples_split: Set to 10 for more conservative splitting
  • min_samples_leaf: Increased to 5 for reliable leaf predictions
  • max_features: 'sqrt' for better feature diversity
Optimization Outcome

The optimized Random Forest achieved the best balance of precision (0.5699), stability (0.3075), and generalization (overfitting gap: 0.1229), making it the most trustworthy choice for Ayurvedic herb-target interaction predictions where safety and reliability are paramount.

Features & Methodology

Data Features
  • Herb Encodings: Numerical representation of Ayurvedic herbs
  • Target Encodings: Numerical representation of biological targets
  • Compound Information: Active compounds in each herb
  • Bioactivity Metrics: Ki, IC50, pKi, pIC50 values
  • Therapeutic Properties: Anti-inflammatory, neuroprotective, etc.
  • Interaction Features: Herb-Target, Compound-Target interactions
  • Quality Scores: Data quality and confidence metrics
Methodology
  • Trustworthy ML: Focus on precision and stability over pure accuracy
  • Biological Relevance: Features designed to capture real biological patterns
  • Cross-Validation: 5-fold CV to ensure model robustness
  • Hyperparameter Optimization: GridSearchCV and RandomizedSearchCV
  • Feature Selection: Mutual information and correlation-based selection
  • Error Analysis: Comprehensive evaluation of prediction errors
  • Model Interpretability: Feature importance and prediction explanations

Usage Guide

User Mode

For general users seeking herbal recommendations based on symptoms:

  1. Navigate to the User page
  2. Enter your symptom (e.g., anxiety, inflammation)
  3. View recommended herbs with predicted effectiveness
  4. Check safety information for each herb
  5. Explore therapeutic properties and mechanisms
Note: Predictions are based on computational models and should not replace professional medical advice.
Researcher Mode

For researchers studying herb-target interactions:

  1. Navigate to the Researcher page
  2. Select or enter an Ayurvedic herb name
  3. View predicted biological targets and interactions
  4. Analyze compounds and pathways
  5. Explore bioactivity scores and confidence metrics
  6. Download results for further analysis
Research Use: These predictions are for research purposes and require experimental validation.

API Documentation

AyurPredict System Class

The core functionality is implemented in the AyurPredictSystem class:

Initialization
system = AyurPredictSystem(model_path, encoder_path, data_path)
Researcher Pathway
result = system.researcher_pathway(herb_name)
# Returns: predicted targets, compounds, pathways, therapeutic effects
Symptom Pathway
result = system.symptom_pathway(symptom)
# Returns: recommended herbs with predicted strength and safety info
Utility Methods
herbs = system.get_herbs()  # List of available herbs
targets = system.get_targets()  # List of available targets
encoded = system.encode(name, key)  # Encode herb/target name
decoded = system.decode(code, key)  # Decode numeric code to name
